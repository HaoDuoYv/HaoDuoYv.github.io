# ✨ Cursor

> AI 驱动的新一代代码编辑器

Cursor 基于 VS Code 构建，深度集成 AI 能力，让开发者用自然语言写代码、理解代码、修复代码。

---

## 🚀 核心能力

| 功能 | 说明 |
|------|------|
| 🤖 AI 代码补全 | 智能预测和补全代码，支持多行上下文感知 |
| 💬 自然语言编程 | 用中文描述需求，自动生成完整代码实现 |
| 📖 代码解释 | 选中任意代码片段，一键获得通俗易懂的解释 |
| 🐛 Bug 修复 | 自动识别代码问题并给出修复建议 |
| 💭 Chat 对话 | 内置 AI 对话窗口，随时提问代码相关问题 |
| 📚 代码库问答 | 基于整个项目上下文回答问题，理解项目全貌 |
